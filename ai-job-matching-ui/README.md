# AI Job Matching UI

Frontend for the AI Resume Screening and Job Matching project.

Built with React and Vite, this app lets users upload a resume, trigger job matching, and view ranked job recommendations returned by the Spring Boot backend.

## Stack

- React 19
- Vite 7
- ESLint

## Scripts

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

Run lint checks:

```bash
npm run lint
```

## Development

Default local URL:

```text
http://localhost:5173
```

This frontend expects the backend API to be running separately.
